policy_id: POL-APPAREL-WINDOW
category: apparel
payment_mode: any
issue_type: wrong_size, return_request
rule_type: exchange
title: Apparel exchange eligibility & size guide
---
Apparel supports size and colour exchanges within the return window when a stock substitute
is available. For a size or fit mismatch — the single largest driver of fashion returns — an
exchange paired with a size guide is the primary deflection lever and is preferred over a
full return when a suitable replacement size is in stock.

If no suitable exchange is available, the customer may take a standard refund to the original
payment method, or a partial refund where they choose to keep the item.
===
---
policy_id: POL-FOOTWEAR-WINDOW
category: footwear
payment_mode: any
issue_type: wrong_size
rule_type: exchange
title: Footwear exchange eligibility
---
Footwear supports size exchanges within the return window when the requested size is in
stock. Exchange with a fit guide is the preferred remedy for a size mismatch; a standard
refund remains available if no exchange can be fulfilled.
